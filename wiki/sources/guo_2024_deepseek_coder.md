# DeepSeek-Coder: When the Large Language Model Meets Programming - The Rise of Code Intelligence

- **元数据**: arXiv | 2024 | **作者**: Daya Guo et al. | **机构**: DeepSeek / Peking University | 相关: [[deepseek_ai]]
- **概述**: 1.3B–33B 开源代码模型系列，在 2T token 项目级代码上训练，采用填空训练任务。
- **新颖概念**: —
- **关键要点**: 1. 项目级代码语料预训练 2. Fill-in-the-Blank 任务 + 16K 窗口 3. 开源全系列代码模型
- **方法/发现**: 从头训练，支持代码生成与填充，在 HumanEval 等基准上匹敌闭源模型。
- **局限/意义**: 推动了代码智能领域的开源生态。
